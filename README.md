# Building
### Building from scratch.
>  You must have
>  - Git
>  - Cargo
>  - Just

<p>Open your terminal of choice and enter</p>

`git clone https://github.com/user93390/Lunara.git` \
`cd Lunara` \
`just build`



### Docker configuration

Build docker image using `just docker-init` \
Run docker by using `just docker-compose`
